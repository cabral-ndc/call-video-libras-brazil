O arquivo `serviceAccountKey.json` contém as credenciais do Firebase Admin SDK e é essencial para que seu **backend** consiga acessar o Firebase Firestore e gerenciar autenticações.  

### 📌 **O que fazer com ele?**  

1️⃣ **Gerar o Arquivo**  
   - Vá até o [Firebase Console](https://console.firebase.google.com/)  
   - Selecione seu projeto  
   - Vá para **Configurações do Projeto** (ícone de engrenagem no menu lateral esquerdo)  
   - Clique em **Contas de Serviço**  
   - Clique em **Gerar nova chave privada**  
   - Baixe o arquivo `serviceAccountKey.json`  

2️⃣ **Mover para o Backend**  
   - Coloque esse arquivo na raiz do seu backend (mesma pasta do `server.js`).  

3️⃣ **Garantir Segurança**  
   **⚠️ Não compartilhe esse arquivo publicamente!**  
   - Adicione `serviceAccountKey.json` ao **`.gitignore`** para evitar subir ao GitHub:
     ```
     serviceAccountKey.json
     ```

4️⃣ **Verificar no `server.js`**  
   - Certifique-se de que o backend está usando o arquivo corretamente:  

   ```javascript
   const admin = require("firebase-admin");
   const serviceAccount = require("./serviceAccountKey.json");

   admin.initializeApp({
     credential: admin.credential.cert(serviceAccount),
   });

   const db = admin.firestore();
   ```

Com isso, seu backend poderá acessar o Firestore e autenticar usuários. 🚀 Se tiver dúvidas, me avise! 😊
